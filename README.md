# 疫情防控-人员健康管理平台（高校、企事业单位通用）

**本项目旨在帮助各高校及企事业单位，在自己的服务器上部署一套私有的人员健康管理系统**

本项目所有的源代码开源，定期开通线上交流活动，帮助大家学习如何部署，如何定制开发。具体信息，请随时关注本项目最新信息。

**本项目近期会在公有云平台搭建多租户的平台（免费开放给无内部部署条件的机构）目前正在落实稳定可靠的云平台资源**
